## Primer Octicons

This project embeds SVG icon files from [Primer Octicons](https://github.com/primer/octicons), stored in `third_party/primer-octicons/` and loaded at compile time via `go:embed`.
The same icon paths also appear inline in generated SVG output files committed to this repository (e.g. `assets/sample-stats.svg`).

- **Version:** 19.23.0
- **Commit:** [`edf68a605cef605d5944d4abb2267cb7a76a4cdd`](https://github.com/primer/octicons/tree/edf68a605cef605d5944d4abb2267cb7a76a4cdd)

The following icons are included:

- `code-review-16.svg`
- `comment-discussion-16.svg`
- `git-commit-16.svg`
- `git-pull-request-16.svg`
- `issue-opened-16.svg`
- `repo-16.svg`
- `sparkles-fill-16.svg`
